## 기술 스택 상세

### 1. 챗봇 채널 — 카카오톡 (무료)
| 구분 | 도구 | 비용 | 설명 |
|------|------|------|------|
| 채널 개설 | 카카오 비즈니스 | 무료 | business.kakao.com |
| 봇 엔진 | 카카오 i 오픈빌더 | 무료 | i.kakao.com |
| 메시지 응답 | 상담톡 Webhook | 무료 | 고객 먼저 시작 |
| 채널 추가 버튼 | 카카오 SDK | 무료 | JS 삽입 가능 |

### 2. 벡터DB — ChromaDB (무료)
| 구분 | 내용 |
|------|------|
| 라이선스 | Apache 2.0 (완전 무료) |
| 설치 | `pip install chromadb` |
| 저장 방식 | 로컬 파일 (./data/chromadb/) |
| 검색 방식 | HNSW 코사인 유사도 |
| 한국어 지원 | TF-IDF 임베딩 호환 |
| 대안 | Qdrant(무료), FAISS(무료), Weaviate(무료) |

### 3. 검색 엔진 — TF-IDF (자체 구현)
| 구분 | 내용 |
|------|------|
| 방식 | Term Frequency-Inverse Document Frequency |
| AI API | 불필요 (순수 수학) |
| 언어 | Node.js 자체 구현 |
| 성능 | FAQ 50문항 검색 < 5ms |
| 정확도 | 키워드 매칭 기반 81%+ |

### 4. 서버 프레임워크 — Express.js (무료)
| 구분 | 내용 |
|------|------|
| 라이선스 | MIT |
| 버전 | v4.18.2 |
| 역할 | Webhook 수신, REST API |
| 미들웨어 | helmet(보안), cors, rate-limit |

### 5. 데이터베이스 — SQLite (무료)
| 구분 | 내용 |
|------|------|
| 라이선스 | Public Domain |
| 용도 | 대화 로그, 미해결 패턴 집계 |
| 라이브러리 | better-sqlite3 (MIT) |
| 용량 | 최대 281TB 지원 |

### 6. 캐시 — node-cache (무료)
| 구분 | 내용 |

| 라이선스 | MIT |
| 방식 | 인메모리 (Redis 불필요) |

### 7. 로거 — Winston (무료)
| 구분 | 내용 |

|------|------|
